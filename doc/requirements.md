# Requirements

## 1. Overview

《Crystal Swarm》是一款 3D 感官轻玩法，玩家用手指制造引力源，把漂浮水晶群聚拢到 3 个能量环中并在 45 秒内完成充能。

## 2. Visual Design

- 画面为竖屏全屏 3D 舞台，逻辑视野以 390px × 680px 为设计基准，外层容器圆角 24px，背景为深空渐变 `#05030f` → `#101837`。
- 主体为 900 个 instanced 水晶，几何体为八面体，基础半径 0.045 到 0.14 world units，颜色在 `#7cf6ff`、`#b692ff`、`#ff74d4`、`#fff4a8` 中按实例混合。
- 水晶材质为高金属/低粗糙度的发光玻璃感，透明度 0.86，实例额外叠加 additive 点精灵光晕，常态屏幕直径约 11px 到 24px，上限 72px。
- 场景中央有一个 3D 引力核心，空闲时半径 0.38，颜色 `#e9ffff`，玩家按下时核心移动到指针投影点并扩大到 0.52。
- 目标环共 3 个，位置分别为左下 `(-1.35,-0.9,0)`、右中 `(1.2,0.05,0)`、上方 `(0,1.05,0)`；每个环半径 0.52，线宽视觉 3px，未充能颜色 `rgba(255,255,255,0.22)`，充能颜色 `#7cf6ff`。
- HUD 顶部左侧显示倒计时，右侧显示分数；字号 12px 标签、30px 数字，字距 0.22em，颜色 `#f7fbff`。
- 底部提示为 11px 大写英文/中文，字距 0.32em，颜色 `rgba(247,251,255,0.62)`；首次按下后淡出。
- 结算卡宽度 318px，高度 260px，背景 `rgba(7,8,20,0.72)`，边框 `1px solid rgba(160,245,255,0.24)`，模糊 20px，圆角 18px。
- 素材清单：`public/img/aigram.svg` 作为右下角 52px 水印；其余视觉全部由 Three.js 几何、材质、灯光和 CSS 生成。

## 3. Game Mechanics

- 初始轮次时长 45 秒，倒计时到 0 后进入结算；没有死亡惩罚。
- 水晶数量固定 900 个；每个水晶有初始位置、速度、轨道相位、目标环归属和随机尺寸。
- 空闲态中，水晶围绕中心做有界群游：半径保持在 1.1 到 2.8 world units，速度上限 0.018 units/frame，自动状态不新增对象。
- 玩家按住屏幕时，指针投影成 z=0 平面上的引力点；水晶受到 `0.0045 / max(distance, 0.22)` 的吸引力，速度上限提高到 0.038 units/frame。
- 玩家松开后，引力强度在 0.55 秒内线性衰减到 0，水晶回到群游状态。
- 每个目标环根据半径 0.62 内的水晶数量计算充能；达到 58 个水晶且保持 1.05 秒后点亮该环。
- 点亮一个环得 300 分；触摸过程中每 0.25 秒按环内水晶密度给 1 到 8 分；全部 3 环点亮额外得 900 分并立即结算。
- Combo 为连续点亮环数量，第二个环显示 `x2 FLOW`，第三个环显示 `x3 PRISM`。
- 最高分使用 `localStorage` 键 `crystal_swarm_best` 持久化。

## 4. Controls

- 开始页点击主按钮进入游戏并解锁音频。
- 游戏中只使用 `pointerdown`、`pointerup` 和低成本 `pointermove` 更新指针坐标；`pointermove` 不创建音频节点、不创建新对象，只更新一个目标向量。
- 按住屏幕吸引水晶，移动手指移动引力点，松开后水晶散开。
- 键盘支持 Space 开始/重新开始，游戏中按住 Space 将引力点锁定在中心。
- 所有交互元素设置 `touch-action: none`、`user-select: none`、`-webkit-tap-highlight-color: transparent`。

## 5. Win / Lose Conditions

- 胜利条件：45 秒内点亮全部 3 个能量环。
- 失败条件：倒计时归零但仍有未点亮能量环；失败只影响结算标题，不扣分。
- 结算界面显示最终分数、历史最高、点亮环数、最大 combo、再来一次、返回首页。
- 三态为开始页、游戏中、结束页，且任意时刻只有一个状态接收 UI 交互。

## 6. Sound Effects

- 开始：sine 波 330Hz → 660Hz 持续 0.18s，随后 triangle 波 880Hz 持续 0.12s，音量 0.05。
- 按下引力：sine 波 110Hz → 180Hz 持续 0.16s，音量 0.035。
- 环充能完成：三音上升 arpeggio，频率 523Hz、659Hz、988Hz，各 0.11s，间隔 0.045s，音量 0.045。
- 全部完成：四音上升，392Hz、523Hz、784Hz、1046Hz，各 0.18s，间隔 0.06s，音量 0.05。
- 时间结束未完成：sawtooth 波 180Hz → 90Hz 持续 0.28s，音量 0.035。
- 按钮点击：square 波 520Hz → 390Hz 持续 0.045s，音量 0.024。
